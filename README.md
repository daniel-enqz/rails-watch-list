## Setup

```shell
 git clone git@github.com:daniel-enqz/rails-watch-list.git
 cd rails-watch-list
 rails db:create db:migrate db:seed:replant
 bundle install
 yarn install
 rails server
```
Open you browser and visit `localhost:3000`
